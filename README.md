
# Image Segmentation Model Comparison

## Overview
This project compares U-Net and DeepLabV3 for semantic image segmentation using the Pascal VOC dataset.

## Key Features
- Implemented using PyTorch
- Applied data augmentation techniques
- Evaluated using mIoU, Pixel Accuracy, FPS

## Results
- Achieved 0.51 mIoU
- Compared performance vs computational efficiency

## Tech Stack
Python, PyTorch, Computer Vision
